## Description

Tech Blog is a CMS-style blog site, where developers can publish their blog posts and comment on other developers' posts as well

## Technologies

-   HTML5
-   CSS3
-   JavaScript
-   Node.js
-   Express.js
-   MySQL
-   Sequelize
-   Express-Session
-   Bootstrap
-   Handlebars
-   Heroku
